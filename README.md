# MMBCU-CVT

手機版 Scooter / MMBCU CVT 傳動模擬器。

## V2.0 修改內容
- 重新建立接近專業調校儀表板的介面。
- 上方分頁已可切換：儀表、傳動、引擎/點火、分析、馬力機、設定/儲存。
- 模式切換已可操作：原廠、Stage 1、Stage 2、Stage 3、自訂設定。
- 車輛選擇改成品牌 → 車型二層選單。
- 修正 MMBCU 歸類為 SYM。
- CVT 動態示意改為 Canvas 即時動畫，會依皮帶位置、轉速、變速比變化。
- 加入 localStorage 儲存/讀取設定。

## 部署
第一層放置 index.html，可直接使用 GitHub Pages。
